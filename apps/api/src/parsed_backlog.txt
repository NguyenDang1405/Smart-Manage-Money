[SMM-AUTH-01] [AUTH] User Registration
Type
Story
Priority
P0
Epic
E-01 – Authentication & User Management
User Story
Là người dùng, tôi muốn đăng ký tài khoản bằng email và mật khẩu để bắt đầu sử dụng ứng dụng.
Acceptance Criteria
✓  Form đăng ký có đủ các trường: Email, Mật khẩu, Xác nhận mật khẩu.

✓  Validation: email đúng định dạng, mật khẩu tối thiểu 8 ký tự.

✓  Sau khi đăng ký thành công, user được redirect vào màn hình onboarding/home.

✓  Nếu email đã tồn tại, hiển thị thông báo lỗi rõ ràng.

✓  Mật khẩu được hash bằng bcrypt trước khi lưu vào DB.

✓  JWT token được tạo và lưu vào AsyncStorage sau khi đăng ký.
Suggested Subtasks
• SUB: Design registration screen UI (Figma → React Native)

• SUB: Implement form validation logic (email & password rules)

• SUB: Build POST /auth/register API endpoint

• SUB: Implement bcrypt password hashing

• SUB: Generate & return JWT token on success

• SUB: Write unit test for registration endpoint
